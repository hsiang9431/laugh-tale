我與勝文不相見已有二年餘了，我最不能忘記的是他的便當。



那年春天，我研究所畢業了，雞排妹也回到前男友身邊了，正是禍不單行的日子，

我從加拿大會來，打算跟著勝文吃香喝辣。

到台北見著勝文，看見滿桌狼籍的廚餘，又想起20萬的紅酒已經被喝光了，

不禁簌簌地流下眼淚。



勝文說，“事已如此，不必難過，好在我們還有壽司便當！”

回家偽裝成高階主管開拍競選廣告，勝文英文不好；又背不起稿子。

這些日子，競選團隊光景很是慘淡，一半為了勝文每天失言，

一半為了以為google被駭客入侵在看洪孟楷耍寶。

洪孟楷耍寶完畢，勝文要到南港內湖點燈，我也要回到台北吃飯，我們便同行。



到台北時，有碰友約去遊逛，勾留了一日；

第二日上午便須摸黑到南港內湖，下午出書誹謗柯P。

勝文因為人肥，本已說定不送我，叫黨裏一個雙面的蕾蕾陪我同去。

他再三囑咐蕾蕾，甚是仔細。



但他終於不放心，怕蕾蕾到了南港和內湖不會游泳；頗躊躇了一會。

其實我那年已二十歲，南港內湖已來往過兩三次，是沒有甚麼要緊的了。

他躊躇了一會，終於決定還是自己送我去。

我兩三回勸他不必去；他只說，“不要緊，你溺水的話可以抓住我！”




我們進了南港內湖，到了碼頭。

我買船票，他忙著照看食物。

食物太多了，得吃掉一些，才可過磅。他便又忙著吃大閘蟹。

我那時真是聰明過分，總覺他動作不大標準，非幫他剝殼不可。

但他終於吃完了大閘蟹；就送我上船。



他給我揀定了靠艙門的一張椅子；我將他給我做的紫茸大衣鋪好座位。

他囑我路上小心，夜裏要吃點宵夜，不要受餓。

又囑托蕾蕾好好照應我。

我心裏暗笑他的迂；他們只認得黨證，托他們直是白托！

而且我這樣大年紀的人，難道還不能自己找東西吃麼？

唉，我現在想想，那時真是太聰明了。



我說道，“勝文，你走吧。”他往車外看了看，

說，“我買幾個638元的壽司便當去。你就在此地，不要走動。”

我看那邊碼頭的柵欄外有幾個賣東西的等著顧客。

走到那邊碼頭，須穿過美食街，正常人須吃下去又吐出來。

勝文是一個胖子，要不受誘惑通過自然要費事些。

我本來要去的，他不肯，只好讓他去。



我看見他含著橘子，腆著大肚子，肥壯的下半身，

蹣跚地走到碼頭邊，慢慢探身下水，尚不大難。

可是他穿過美食街，要回來這邊碼頭，就不容易了。

他用兩手抓著壽司便當，兩腳再夾住披薩；

他肥胖的身子向上微挺，顯出努力想再吃兩口包子的樣子。

這時我看見他手裏吃剩的便當，我的淚很快地流下來了。

我趕緊拭乾了淚，怕他看見，也怕別人看見。



我再向外看時，他已咬了朱紅的橘子往回走了。



過碼頭時，他先將橘子散放在地上，自己慢慢入水，再咬起橘子游泳。

到這邊時，我趕緊去攙他。他和我走到船上，將橘子一股腦兒放在我的皮大衣上，

然後作他的飯後甜點。



於是勝文吃光手中的壽司便當和橘子，心裏很輕鬆似的，過一會說，

“我好像吃八分飽了，到那邊來信！”我望著他走出去。

他走了幾步，回過頭看見我，說，“進去吧，東西早被我吃完了。”

等他的背影混入來來往往的人裏，再找不著了，我便進來坐下，我的眼淚又來了。



近幾年來，勝文和我都是東奔西走，連家光景是一日不如一日。

他少年出外跑玩伴女郎趴，住川普大樓，吃了許多大餐。

哪知老境卻如此頹唐！他觸目傷懷，自然食慾不能自已。

情鬱於中，自然要發之於外；一天沒得吃個八餐便往往觸他之怒。他待我漸漸不同往日。

但最近兩年不見，他終於忘卻我的不好，

只是惦記著我，惦記著我當年根本就沒吃到的壽司便當。



我北來後，他寫了一封信給我，信中說道，

“我身體平安，惟肚子餓得利害，舉箸提筆，諸多不便，大約又到了吃飯時間矣。”

我讀到此處，在晶瑩的淚光中，又看見那肥胖的，雙手拿著便當，腳上夾著批薩的背影。

唉！我不知何時才能好好吃一頓！
